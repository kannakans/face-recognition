# face-recognition

This is a face recognition model.The samples of the face are collected in run time by using the gather_selfies.py file.
The faces are captured by pressing the 'c' button and it is exited by using the 'q' button.
Then the faces are recognized using the train_recognizer.py file and tested using the security_cam.py file

This is written purely in python with the help of opencv-python.
